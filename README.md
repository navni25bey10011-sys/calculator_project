# Basic Calculator Project 

## Overview
This is a simple modular Python-based calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.  
The project is designed according to the "Build Your Own Project" guidelines and demonstrates modularity, input validation, and clean architecture.


## Features
- Addition  
- Subtraction  
- Multiplication  
- Division  
- Error handling for invalid inputs  
- Division-by-zero protection  
- Modular file structure  
- Easy to extend  


## Technologies Used
- Python 3.x  
- VS Code  
- Modular Architecture
